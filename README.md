# AI-powered-greenhouse-control
A greenhouse control system which uses sensors integrated with an STM32 bluepill microcontroller. An AI plugin gives optimum values specific to each plant for best control enviromnets. The system consists of a temperature and humidity sensor as well as a soil moisture sensor. The blue pill sends realtime values to the control hub(webpage UI). The AI gives optimum temp, humidity, and soil moisture values based on the plant type, location, and time of day. Based on these value the UI gives istructions for the gardener to follow. Can also be fully automated with accuators.
